# My tickets

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```


## Dir structure

```
├───@types     // TS typings
├───components // A reusable piece of code across page
├───layouts    // Common page layout
├───pages      // Home, login...
├───plugins    // Vuetify 
└───services   // Backend calls
    └───mocks
```
